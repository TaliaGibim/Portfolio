My neural network from scratch

This neural network was built using only NumPy, without any pre-built packages. The knowledge used for this project was gained through this Coursera course by Andrew Ng. In this project, I created a neural network for the 'Hello World' of data science: the digit recognition problem.

Using NumPy, Pandas, and object-oriented programming, I created a class that contains my neural network. Using Pickle, this neural network was exported. A grid search was performed to find the best architecture and hyperparameters.

Results

In the graph below, the results of train accuracy for each model are summarized. It is possible to see that the best architecture is the one with a single layer with 10 neurons.



In more detail, we can closely examine the results from this architecture to observe the impact of both the scaler and the alpha parameter. There is no significant difference between using a scaler of 0.005 and 0.01; however, increasing the interactions clearly improves the model's accuracy, as expected. The best alpha appears to be 1. To confirm these findings, an ANOVA analysis was performed.

ANOVA

Using the Kolmogorov-Smirnov test to check the normality of the distributions, the results showed that all three models had p-values greater than 0.05, indicating that their distributions are not significantly different from a normal distribution.



Given this, we proceeded with ANOVA, which is appropriate for normally distributed data. The ANOVA test revealed a test statistic of 88.91 and a p-value of 2.38e-17. Since the p-value is less than 0.05, we can reject the null hypothesis, suggesting that there are significant differences between the models. The Tukey's HSD test was then performed to identify which specific models differ.




Lastly, the effect size (η²) was calculated to be 1.68, indicating that the differences between the models are large and represent a strong effect. In conclusion, the data for each model is normally distributed, and significant differences were found between the models according to the ANOVA test. Model 1 was significantly different from Model 3 based on the Tukey's HSD test, and the large effect size suggests the differences are meaningful.


Since the choice between Model 1 and Model 2 does not significantly affect the outcome, the model with the highest training accuracy was selected.

The neural network implemented on this webpage uses an alpha value of 1, a scalar of 0.005, 80 interactions, and a single layer with 10 neurons. This model achieved 90.1% accuracy on the training set and 90.7% accuracy on the test set.
